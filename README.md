Pragati College Event Registration System

A modern, responsive web application for managing and registering college events. This project is built using Next.js, TypeScript, and reusable UI components, providing a clean and interactive user experience.

---

Features

* Professional college event interface
* Event listing and browsing system
* Interactive registration forms
* Reusable UI components (buttons, cards, dialogs, etc.)
* Fully responsive design
* Fast performance using Next.js
* Toast notifications and alerts
* Clean and modular code structure

---

Tech Stack

* Next.js – React framework
* TypeScript – Type-safe JavaScript
* CSS / Global Styles
* Component-based UI architecture

---

Project Structure

```
  images

```

---

How to Run the Project

1. Install dependencies
   npm install

2. Run development server
   npm run dev

3. Open in browser
   [http://localhost:3000](http://localhost:3000)
